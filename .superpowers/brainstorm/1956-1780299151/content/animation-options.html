배정 시작 애니메이션 — 3가지 방식

주최자 PC에서 "배정 시작" 버튼을 누르면 어떤 연출을 원하시나요?

A

드럼롤 셔플

모든 참가자 이름이 빠르게 섞이다가 차례로 속도가 줄어들며 팀이 하나씩 잠긴다. 슬롯머신처럼 팀 1 → 팀 2 → … 순서로 순차 확정.

🎰 팀 1 배정 중...
[ 김철수 ] ↔ [ ??? → ??? → 홍길동 ✓ ]
🎰 팀 2 배정 중...
[ 이영희 ] ↔ [ 최지수 ✓ ]
✔ 긴장감·기대감 최고 / 팀 하나씩 집중
B

카드 플립 + 페어링

참가자 카드가 뒤집힌 채로 화면에 펼쳐진다. 배정 시작 시 카드가 2장씩 동시에 뒤집히며 팀이 공개된다. 모든 팀이 같은 시간에 서서히 드러남.

C

A + B 혼합 (드럼롤 → 전체 공개)

먼저 드럼롤로 긴장감을 고조시키고, 모든 팀이 배정되면 마지막에 전체 결과 카드가 화면 가득 펼쳐지는 피날레 연출.

1단계: 🎰 드럼롤 (팀별 순차 확정)
2단계: 🎉 전체 결과 카드 펼침
"Tournament Draft — 오늘의 파트너" 전광판 연출
✔ 긴장감 + 피날레 모두 / 구현 복잡도 높음
